# Reward-Hub-App

#### This app is made for user to earn coins and get money.
#### Google Sign In, Spin Wheel, Scratch Card, Recaptcha, Invite Link.
#### used firebase as backend and implemented applovin ads.
#### Technologies : Java , XML , Firebase.
